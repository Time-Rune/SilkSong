## 核心机制

- 打怪 掉落 经验条 & 金币
- 升级 给 点数
- build 强身健体 消耗 点数 ---> 在凳子上
  - 血量 HP
  - 蓝条 Silk
  - 攻击力 Attack
- skill tree 技能树 消耗 金币 ---> 在村庄 的 teacher处
- 击败最终Boss => 获得胜利
